# Swift JSON Formatter

Swift JSON Formatter is a library and CLI for deterministic JSON formatting.

## Install

Add the package to your `Package.swift` dependencies:

```swift
dependencies: [
    .package(url: "https://github.com/swift-universal/swift-json-formatter.git", from: "0.1.0")
]
```

## CLI

Run the CLI from the package root:

```bash
swift run --package-path . swift-json-formatter --help
```
